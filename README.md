## Redux basic tutorial

### Setup

```base
sudo npm install -g create-react-app
create-react-app test-react-redux
```
```base
cd test-react-redux
npm install --save redux react-redux
```   

### Installation

```bash
git clone https://github.com/jnguyen095/redux.git
cd redux
npm install
npm start
open http://localhost:3000
```
